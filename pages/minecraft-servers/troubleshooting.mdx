# Troubleshooting

This guide covers the most common issues you may encounter while managing your Minecraft server.

If your issue isn't listed here, our support team is always available through Discord and the Client Area.

---

## My Server Won't Start

If your server refuses to start:

- Check the Console for error messages.
- Verify that your server software is compatible with your Minecraft version.
- Remove any recently installed plugins or mods.
- Verify your Startup configuration.
- Restore a backup if necessary.

---

## I Can't Join My Server

If you're unable to connect:

- Verify that your server is online.
- Confirm you're using the correct server address.
- Check that you're using the correct port (if using a Shared IP).
- Verify that your Minecraft version matches the server version.

---

## Connection Timed Out

This usually indicates a connection problem.

Possible causes include:

- Server still starting.
- Incorrect IP or port.
- Local network issues.
- Temporary connectivity problems.

Wait a few moments and try again.

---

## My Server Crashes After Installing a Plugin

Recently installed plugins are one of the most common causes of startup failures.

Try:

- Removing the newest plugin.
- Updating the plugin.
- Checking plugin compatibility.
- Reviewing console errors.

---

## High RAM Usage

High RAM usage isn't always a problem.

Minecraft intentionally uses available memory to improve performance.

However, if your server begins crashing:

- Reduce unnecessary plugins.
- Lower the player count.
- Increase your plan if necessary.

---

## High CPU Usage

CPU usage may increase during:

- World generation.
- Large explosions.
- Redstone machines.
- Heavy farms.
- Large numbers of entities.

Optimizing your server software can significantly improve performance.

---

## My World Won't Load

Possible causes include:

- World corruption.
- Incompatible Minecraft version.
- Missing mods.
- Plugin conflicts.

Restore a recent backup if available.

---

## SFTP Connection Failed

If you cannot connect through SFTP:

- Verify your username.
- Verify your password.
- Check the host and port.
- Make sure the server has finished deploying.

---

## Database Connection Errors

If a plugin cannot connect to MySQL:

- Verify the database credentials.
- Confirm the database exists.
- Restart the server after updating the configuration.
- Check the plugin configuration for typing mistakes.

---

## Server Shows Offline

Sometimes the panel may temporarily display your server as offline while it is starting.

Wait a few moments and refresh the page.

If the issue persists, review the Console for startup errors.

---

## Server Is Stuck on "Installing"

If your server remains in the **Installing** state for several minutes:

- Refresh the control panel.
- Wait for the installation process to complete.
- Avoid reinstalling the server multiple times.

If the installation does not complete after several minutes, contact our support team.

---

## Server Is Stuck on "Starting"

If the server never reaches the **Running** state:

- Open the Console.
- Look for Java errors.
- Verify installed plugins or mods.
- Restart the server.
- Restore a recent backup if necessary.

---

## Still Need Help?

If none of these solutions resolve your issue, please open a support ticket through Discord or the Client Area.

When requesting support, include:

- The server software.
- Minecraft version.
- Console errors.
- Screenshots (if applicable).
- Steps you've already tried.
