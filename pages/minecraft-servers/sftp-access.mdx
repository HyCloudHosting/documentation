# SFTP Access

SFTP (Secure File Transfer Protocol) allows you to securely access your server files using a desktop application instead of the web-based File Manager.

SFTP is the recommended method for transferring large files, managing multiple files at once, or performing extensive file operations.

Unlike traditional FTP, SFTP encrypts all communication between your computer and the server.

---

## When Should You Use SFTP?

We recommend using SFTP when you need to:

- Upload large world files.
- Install multiple plugins.
- Upload modpacks.
- Transfer resource packs.
- Download complete server backups.
- Move many files at once.
- Manage files more efficiently than the web File Manager.

For small edits or quick uploads, the File Manager is usually sufficient.

---

## Recommended SFTP Clients

HyCloud Hosting is compatible with any SFTP client.

Some popular options include:

- FileZilla
- WinSCP
- Cyberduck

Choose the client that best fits your operating system and personal preference.

---

## Finding Your SFTP Credentials

Your SFTP connection information is available directly in the Pterodactyl Control Panel.

To view it:

1. Log in to the Control Panel.
2. Open your server.
3. Navigate to **Settings**.
4. Locate the **SFTP Details** section.

You'll find information such as:

- Server Address
- SFTP Port
- Username

Your panel password is also your SFTP password.

---

## Connecting to Your Server

To establish an SFTP connection:

1. Open your preferred SFTP client.
2. Enter the SFTP Host.
3. Enter the Port displayed in the panel.
4. Enter your Username.
5. Enter your Panel Password.
6. Connect to the server.

Once connected, you'll be able to browse and manage your server files.

---

## Uploading Files

After connecting successfully, you can:

- Drag and drop files.
- Upload folders.
- Replace existing files.
- Download server files.
- Rename files.
- Delete files.
- Move folders.

Most SFTP clients support drag-and-drop for quick file transfers.

---

## Large File Transfers

SFTP is the recommended method for uploading:

- Large Minecraft worlds.
- Modpacks.
- Plugin collections.
- Resource packs.
- Backup archives.

Large uploads are generally faster and more reliable than using the web File Manager.

---

## Security Recommendations

To help keep your account secure:

- Never share your SFTP credentials.
- Disconnect when you're finished.
- Only connect using trusted networks.
- Keep your SFTP client updated.

Since SFTP uses your panel password, changing your panel password will also change your SFTP password.

---

## Troubleshooting Connection Issues

If you cannot connect using SFTP:

- Verify that your username is correct.
- Verify your panel password.
- Confirm the server address and port.
- Ensure your server is active.
- Check that your firewall or antivirus is not blocking the connection.

If the problem persists, contact our support team.

---

## Need Help?

If you're having trouble connecting via SFTP, please provide the following when opening a support ticket:

- The SFTP client you're using.
- Any error message displayed.
- A screenshot of the connection settings (without revealing your password).

This information helps us diagnose the issue more quickly.
