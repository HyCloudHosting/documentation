# Server Management

Every Minecraft server hosted by HyCloud Hosting is managed through the Pterodactyl Control Panel.

The panel provides a modern web interface that allows you to control nearly every aspect of your server without requiring command-line knowledge.

Whether you're starting your first server or managing a large community, the control panel gives you the tools you need to keep your server running smoothly.

---

## What You Can Do

Using the control panel, you can:

- Start, stop and restart your server.
- View the live console.
- Upload, download and manage files.
- Create and restore backups.
- View startup variables and server configuration.
- Monitor CPU, RAM and disk usage.
- Schedule automated tasks.
- Manage databases.
- Manage network allocations.
- Reinstall your server.
- View activity logs.

Most day-to-day server administration can be performed directly from the web panel.

---

## Main Sections

The navigation menu is divided into several sections, each designed for a specific purpose.

### Console

Monitor your server in real time, execute commands and review startup logs.

---

### Files

Upload, edit, rename, move and delete files directly from your browser.

You can also create folders, upload plugins or mods, and extract compressed archives.

---

### Startup

The **Startup** page displays the startup variables and command used to launch your server.

At this time, startup variables are **read-only** and cannot be modified through the control panel.

If you need to change a startup setting, please contact our support team through Discord or the Client Area.

---

### Schedules

Automate repetitive tasks such as:

- Restarting your server
- Creating backups
- Running console commands

Schedules help reduce manual maintenance.

---

### Backups

Create manual backups whenever needed.

Backups can later be restored directly from the control panel.

We strongly recommend creating a backup before making major changes to your server.

---

### Network

View your server's IP address and assigned ports.

Additional allocations may be available depending on your hosting plan.

---

### Databases

Create and manage MySQL databases for plugins and server software that require external storage.

Database availability depends on your hosting plan.

---

### Activity

Review recent actions performed on your server.

This includes administrative operations such as server startups, reinstalls and configuration changes.

---

## Monitoring Resources

The control panel continuously displays your server's resource usage, including:

- CPU usage
- Memory (RAM)
- Disk usage
- Network activity

Monitoring these values can help identify performance issues before they affect players.

---

## Best Practices

To keep your server running smoothly, we recommend:

- Restart your server after installing plugins or mods.
- Create backups before making important changes.
- Monitor RAM usage regularly.
- Remove unused files and plugins.
- Keep your server software updated whenever possible.
