# File Manager

The File Manager allows you to access and manage your server files directly from your web browser.

Whether you're uploading plugins, editing configuration files, or organizing your server folders, the File Manager provides an easy way to manage everything without requiring external software.

---

## What You Can Do

Using the File Manager, you can:

- Browse your server files and folders.
- Upload new files.
- Download existing files.
- Create folders.
- Create new files.
- Rename files and folders.
- Move files.
- Copy files.
- Delete files and folders.
- Edit text-based files directly in your browser.
- Compress files into archives.
- Extract ZIP archives.
- Change file permissions (when supported).

---

## Uploading Files

To upload files:

1. Open **File Manager**.
2. Navigate to the destination folder.
3. Click **Upload**.
4. Select one or more files from your computer.
5. Wait for the upload to finish.

Large uploads may take several minutes depending on your internet connection.

> **Tip**
>
> For large uploads, such as world files or modpacks, we recommend using **SFTP** instead of the web File Manager for a faster and more reliable transfer.

---

## Editing Files

Many server configuration files can be edited directly from the browser.

Examples include:

- `server.properties`
- `paper.yml`
- `spigot.yml`
- `bukkit.yml`
- Plugin configuration files
- JSON configuration files
- YAML configuration files

After saving your changes, you may need to restart your server for them to take effect.

---

## Creating Folders

To create a new folder:

1. Open the desired directory.
2. Click **New Folder**.
3. Enter a folder name.
4. Confirm the creation.

Folders help keep your server organized, especially when managing plugins or custom content.

---

## Creating Files

You can also create new text files directly from the File Manager.

This is useful for:

- Configuration files
- JSON files
- YAML files
- Text notes
- Custom scripts (if supported)

---

## Compressing Files

The File Manager allows you to compress multiple files or folders into a ZIP archive.

This is useful when:

- Downloading several files at once.
- Creating manual backups.
- Organizing server data.

---

## Extracting Archives

ZIP archives can be extracted directly from the File Manager.

This is commonly used when:

- Installing plugins.
- Uploading resource packs.
- Importing world files.
- Uploading server templates.

If the archive contains many files, extraction may take a few moments.

---

## Downloading Files

Individual files and compressed archives can be downloaded directly to your computer.

Downloading important configuration files before making major changes is considered a good practice.

---

## Deleting Files

Files and folders can be permanently deleted from the File Manager.

> **Warning**
>
> Deleted files cannot be recovered unless you have a backup available.

Always verify that you are deleting the correct files before confirming the action.

---

## Best Practices

To keep your server organized, we recommend:

- Create backups before editing important files.
- Organize plugins inside their default folders.
- Remove unused files regularly.
- Avoid renaming system folders.
- Keep configuration files properly formatted.

---

## Common Mistakes

Avoid the following:

- Deleting important configuration files.
- Uploading plugins while the server is running.
- Editing files without creating a backup first.
- Extracting archives into the wrong directory.
- Uploading unsupported file types.

---
