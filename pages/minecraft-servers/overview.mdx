# Minecraft Servers

HyCloud Hosting provides high-performance Minecraft server hosting designed for communities of all sizes.

Whether you're creating a small survival server for friends or managing a large public network, our platform is built to deliver reliable performance, powerful management tools, and an easy deployment experience.

---

## What We Offer

Every Minecraft server includes everything you need to get started:

- High-performance hardware
- DDoS protection
- NVMe SSD storage
- Full file access
- One-click server management
- Scheduled backups
- Custom startup settings
- FTP access
- Automatic reinstall support
- Full access to the Pterodactyl control panel

Our infrastructure is designed to provide stable performance with minimal downtime.

---

## Server Software

During the ordering process, you will be asked to choose the software you want your Minecraft server to run.

Your selected software will be automatically installed before the server is provisioned, allowing you to start using your server immediately after deployment.

If you need a different software later, you can reinstall your server or contact our support team for assistance if necessary.

## Server Management

Every Minecraft server is managed through the Pterodactyl Control Panel.

Once your server has been deployed, you can:

- Start, stop and restart your server.
- Access the live console.
- Upload and manage files.
- Create scheduled tasks.
- Configure startup variables.
- Create and restore backups.
- Monitor resource usage.
- Reinstall your server when needed.

Your selected server software is automatically installed during deployment, allowing you to begin playing as soon as your server is ready.
