# Activity Logs

The Activity Logs page provides a history of administrative actions performed on your Minecraft server.

These logs help you understand what changes have been made, when they occurred, and who performed them.

Activity Logs are useful for troubleshooting issues, auditing server management, and reviewing recent changes.

---

## What Is Recorded?

The Activity Logs may include actions such as:

- Server starts and stops.
- Server restarts.
- Backup creation and restoration.
- File operations.
- Schedule execution.
- Database management.
- Startup configuration changes.
- Subuser actions.
- Server reinstalls.

The available events may vary depending on the action performed.

---

## Viewing Activity Logs

To access the Activity Logs:

1. Open your Minecraft server.
2. Navigate to **Activity**.
3. Review the list of recorded events.

Each entry includes information about the performed action and when it occurred.

---

## Why Are Activity Logs Useful?

Activity Logs can help you:

- Determine when a change was made.
- Verify who performed an action.
- Investigate unexpected behavior.
- Review administrative activity.
- Troubleshoot configuration issues.

They're especially useful when multiple administrators manage the same server.

---

## Understanding the Entries

Each log entry represents a single action performed through the control panel.

Depending on the event, an entry may include:

- The action performed.
- The user who performed it.
- The date and time.
- Additional details about the operation.

---

## Best Practices

We recommend:

- Review Activity Logs after making important changes.
- Check the logs when troubleshooting issues.
- Periodically review activity if your server has multiple administrators.
- Use Activity Logs together with backups to better understand changes made over time.

---

## Limitations

Activity Logs are intended to record administrative actions performed through the control panel.

They do **not** replace:

- Minecraft server logs.
- Console output.
- Plugin logs.

If you're troubleshooting gameplay or plugin-related issues, check the **Console** or the relevant log files in your server.

---
