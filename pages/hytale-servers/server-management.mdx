# Server Management

Your Hytale server is managed through the HyCloud Hosting Control Panel, powered by Pterodactyl.

The control panel provides all the tools needed to operate your server, monitor its status, and manage its files from any modern web browser.

---

## Power Controls

The control panel provides several power options:

- **Start** — Starts the server.
- **Stop** — Safely shuts down the server.
- **Restart** — Restarts the server.

Always use these controls instead of forcefully closing the server whenever possible.

---

## Live Console

The Console allows you to monitor your server in real time.

From the console you can:

- View server startup logs.
- Monitor warnings and errors.
- Execute server commands.
- Observe player activity (when available).
- Troubleshoot startup issues.

The console is one of the most useful tools for diagnosing problems.

---

## Resource Monitoring

The control panel continuously displays your server's resource usage.

You can monitor:

- CPU usage
- Memory (RAM) usage
- Disk usage
- Network activity
- Server uptime

Monitoring these metrics helps identify performance issues before they affect gameplay.

---

## File Management

All server files can be managed directly through the built-in File Manager.

From there you can:

- Upload files.
- Download files.
- Rename files and folders.
- Delete files.
- Edit configuration files.
- Create new folders.

For larger transfers, you can also use SFTP if it becomes available for your service.

---

## Backups

Creating regular backups is strongly recommended.

Backups allow you to restore your server if:

- Files become corrupted.
- A configuration mistake is made.
- You accidentally delete important data.

Always create a backup before making significant changes.

---

## Server Status

The control panel displays your server's current status.

Possible statuses include:

| Status | Description |
|---------|-------------|
| Running | Your server is online and accepting connections. |
| Starting | The server is currently starting. |
| Stopping | The server is shutting down safely. |
| Offline | The server is not running. |
| Installing | The server is being deployed or reinstalled. |

---

## Best Practices

To keep your server running smoothly, we recommend:

- Restarting the server after major configuration changes.
- Reviewing the console regularly for errors.
- Creating backups before making important changes.
- Monitoring resource usage during gameplay.
- Avoiding unnecessary force restarts.

---

HyCloud Hosting will continue expanding the available management tools as new official server capabilities become available.
