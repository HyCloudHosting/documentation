# Troubleshooting

This guide covers some of the most common issues you may encounter while managing your Hytale server.

If your problem is not listed here, our support team is available through Discord and the Client Area.

---

## My Server Won't Start

If your server fails to start:

- Review the Console for error messages.
- Verify that the server deployment completed successfully.
- Restart the server.
- Restore a recent backup if necessary.

If the problem continues, contact our support team.

---

## Server Is Stuck on "Installing"

If your server remains in the **Installing** state for an extended period:

- Wait a few minutes for the deployment to finish.
- Refresh the control panel.
- Avoid reinstalling the server multiple times.

If the installation never completes, please open a support ticket.

---

## Server Is Stuck on "Starting"

If your server never reaches the **Running** state:

- Open the Console.
- Review any startup errors.
- Restart the server.
- Restore a recent backup if you recently modified server files.

---

## I Can't Connect to My Server

If you're unable to join your server:

- Verify that the server is online.
- Confirm you're using the correct server address.
- Ensure the server has fully started.
- Check your internet connection.

If the issue persists, contact our support team.

---

## Control Panel Is Unavailable

If you cannot access the Control Panel:

- Verify that you can log in to the Client Area.
- Try refreshing the page.
- Clear your browser cache if necessary.
- Wait a few minutes and try again.

Temporary maintenance may occasionally affect panel availability.

---

## File Upload Failed

If a file upload does not complete:

- Verify that the file is not corrupted.
- Refresh the File Manager.
- Try uploading the file again.
- Upload smaller files individually if necessary.

---

## Backup Failed

If a backup cannot be created:

- Wait a few moments and try again.
- Verify that your server is responding normally.
- Avoid creating multiple backups simultaneously.

If the problem continues, contact support.

---

## Server Performance Issues

If your server feels slow or becomes unresponsive:

- Restart the server.
- Monitor CPU and RAM usage.
- Avoid unnecessary file modifications while the server is running.
- Check the Console for warnings or errors.

---

## Still Need Help?

If none of the solutions above resolve your issue, please open a support ticket through Discord or the Client Area.

When contacting support, include:

- Your server name.
- A description of the problem.
- Any error messages shown in the Console.
- Screenshots, if available.
- The steps you've already tried.

Providing this information helps our team diagnose and resolve your issue more quickly.
