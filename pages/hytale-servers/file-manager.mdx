# File Manager

The File Manager allows you to browse, upload, edit, download, and organize your Hytale server files directly from the control panel.

It provides a convenient way to manage your server without requiring additional software.

---

## Opening the File Manager

To access your server files:

1. Log in to the Control Panel.
2. Select your Hytale server.
3. Click **Files** from the navigation menu.

The File Manager will display your server's directory structure.

---

## Uploading Files

To upload files:

1. Open the destination folder.
2. Click **Upload**.
3. Select one or more files from your computer.
4. Wait for the upload to complete.

Large uploads may take several minutes depending on your internet connection.

---

## Editing Files

Many configuration files can be edited directly from the browser.

To edit a file:

1. Open the desired file.
2. Make your changes.
3. Click **Save**.

Always verify your changes before restarting the server.

---

## Creating Files and Folders

You can organize your server by creating new files and directories.

Available actions include:

- Create new folders.
- Create new files.
- Rename existing files.
- Move files between directories.
- Delete unwanted files.

Keeping your files organized makes server management much easier.

---

## Downloading Files

Any file stored on your server can be downloaded to your computer.

This is useful for:

- Creating local backups.
- Editing files offline.
- Reviewing logs.
- Keeping copies of important configuration files.

---

## Compressing Files

The File Manager supports compressing multiple files into a single archive.

This can be useful when:

- Downloading large directories.
- Organizing server files.
- Preparing files for transfer.

Supported archive formats may vary depending on the control panel version.

---

## Extracting Archives

Compressed archives can also be extracted directly from the File Manager.

Supported archive formats typically include:

- ZIP
- TAR
- TAR.GZ

This makes it easy to upload and extract multiple files at once.

---

## Best Practices

We recommend:

- Create a backup before modifying important files.
- Keep your server files organized.
- Delete files you no longer need.
- Download important configuration files periodically.
- Review server logs when troubleshooting.

---

## Current Limitations

The File Manager is intended for general server management.

Very large file transfers may be faster when using other transfer methods if they become available in the future.

---
