# Accessing the Control Panel

Every Hytale server is managed through the HyCloud Hosting Control Panel, powered by Pterodactyl.

The control panel allows you to start and stop your server, monitor its performance, manage files, create backups, and access the server console from any modern web browser.

---

## Accessing the Panel

After your Hytale server has been deployed:

1. Log in to the Client Area.
2. Navigate to **Services**.
3. Select your Hytale server.
4. Click **Login to Panel**.

You will be automatically redirected to the control panel without needing to enter your credentials again.

---

## First-Time Access

When accessing the control panel for the first time, we recommend setting your own password.

To do this:

1. Open the control panel.
2. Click your account avatar.
3. Select **Account**.
4. Choose **Reset Password**.
5. Follow the instructions sent to your email.

Using your own password allows you to access the control panel directly whenever needed.

---

## Linking External Accounts

For easier sign-in, you can link external authentication providers to your control panel account.

Currently supported providers include:

- Google
- Discord
- GitHub

To manage linked accounts, visit:

**https://panel.hycloudhosting.com/account/**

After linking an account, you can use it to sign in to the control panel without entering your password.

---

## What You Can Do

Once inside the control panel, you can:

- Start, stop, and restart your server.
- View the live server console.
- Manage server files.
- Upload files using the File Manager.
- Create and restore backups.
- Monitor CPU, RAM, and disk usage.
- View server activity.

---

## Security Recommendations

To help keep your account secure, we recommend:

- Using a strong, unique password.
- Enabling an external login provider such as Google or GitHub.
- Never sharing your account credentials.
- Logging out when using a shared computer.

---

## Having Trouble Accessing the Panel?

If you're unable to access your control panel:

- Verify that your Client Area account is active.
- Confirm that your server has finished deploying.
- Reset your password if necessary.
- Contact our support team if the issue persists.

---

## Need Help?

If you experience any problems accessing your control panel, our support team is available through Discord and the Client Area.

For the fastest assistance, we recommend opening a Discord support ticket.
